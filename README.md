# 💰 Live Expense Tracker

A lightweight, fully responsive personal finance dashboard that helps users track their income, monitor expenses, and view their real-time net balance.

🔗 **[Live Demo Link](YOUR_NETLIFY_OR_GITHUB_PAGES_LINK_HERE)**

## ✨ Features

- **Dynamic Calculation:** Instantly updates your running total balance, overall income, and total expenses as data is input.
- **Persistent Data:** Leverages browser `localStorage` to ensure financial logs are retained even after refreshing or closing the browser window.
- **Visual Feedback:** Color-coded transaction feeds (Green for credit, Red for debit) for intuitive visual tracking.
- **Transaction History management:** Allows users to easily add new logs or delete historical records instantly via dynamic DOM manipulation.

## 🛠️ Tech Stack Used

- **HTML5:** Structured semantic markup (`<main>`, `<section>`, `<form>`).
- **CSS3:** Custom UI layout featuring flexbox layouts, CSS variables, and interactive hover transitions.
- **Vanilla JavaScript:** Event-driven functionality, custom input validation, array mapping/reducing, and browser database integration.

## 🧠 Key Learning Outcomes

- Advanced my understanding of the JavaScript `Array.prototype.reduce()` and `filter()` methods for math metrics.
- Mastered seamless synchronization between local JSON storage arrays and the active Document Object Model (DOM).
